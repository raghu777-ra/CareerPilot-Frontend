# CareerPilot Frontend

CareerPilot is a modern AI-Powered Job Portal frontend built with React. It provides an intuitive interface for candidates, recruiters, and administrators to manage the job recruitment process.

## 🚀 Features

- User Authentication (JWT)
- Candidate Dashboard
- Recruiter Dashboard
- Admin Dashboard
- Browse & Search Jobs
- Job Details Page
- Save Jobs
- Apply for Jobs
- Profile Management
- Responsive Design
- Modern Glassmorphism UI

## 🛠️ Tech Stack

- React
- Vite
- React Router
- Axios
- Bootstrap 5
- CSS3
- JWT Authentication

## 📁 Project Structure

```
src/
├── components/
├── pages/
├── services/
├── context/
├── layouts/
├── assets/
└── css/
```

## ⚙️ Installation

```bash
git clone <repository-url>
cd CareerPilot-Frontend
npm install
npm run dev
```

## 🌐 Backend

This frontend communicates with the CareerPilot Spring Boot backend using REST APIs.
